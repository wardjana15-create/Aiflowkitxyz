Why Automate Client Onboarding?

As a freelancer, your time is money. Yet most of us waste hours on:

  • Sending contracts
  • Creating invoices
  • Writing welcome emails
  • Setting up client folders

This workflow automates it all — from first Calendly booking to final invoice.

🛠️ Tools You’ll Need

  • Calendly (Free Tier) — for scheduling
  • Zapier (Free Tier) — for automation
  • Notion AI (Free Tier) — for document generation
  • Stripe (Free to start) — for invoicing
  • Gmail — for emails

All tools have free plans. No payment required to start.

Step 1: Set Up Calendly Trigger

In Calendly, go to “Event Types” → Create new → Name: “Client Onboarding Call”.

Under “Notifications” → “Webhooks” → Add Zapier webhook URL (get this in Step 2).

Save. Now every booking triggers the automation.

Step 2: Create Zapier Automation

Go to zapier.com → “Create Zap”.

Trigger: “Calendly” → “New Invitee” → Connect your Calendly account.

Test trigger → Should pull client name, email, meeting time.

Step 3: Generate Contract in Notion

Add Action: “Notion” → “Create Page”.

Connect Notion → Choose database “Clients”.

Map fields:

  • Title: {{invitee_first_name}} {{invitee_last_name}} Contract
  • Client Name: {{invitee_first_name}} {{invitee_last_name}}
  • Email: {{invitee_email}}
  • Contract Template: Paste your standard contract text

Turn on Notion AI → “Summarize” → “Generate first draft”.

Step 4: Send Invoice via Stripe

Add Action: “Stripe” → “Create Customer” → Map email + name.

Add Action: “Stripe” → “Create Invoice” → Set amount, due date, description.

Enable “Auto-send invoice” → Client gets email instantly.

Step 5: Send Welcome Email via Gmail

Add Action: “Gmail” → “Send Email”.

To: {{invitee_email}}

Subject: “Welcome! Your contract + invoice are ready”

Body: “Hi {{invitee_first_name}}, thanks for booking! Your contract is here: [Notion Link]. Your invoice: [Stripe Link]. Let me know if you have questions!”
