Why Start with AI Automation?

If you’re overwhelmed by repetitive tasks — emails, data entry, scheduling, content formatting — AI automation is your escape hatch.

You don’t need to be a techie. You don’t need to spend money. You just need to start small.

This guide walks you through your first 30-minute automation — step by step.

🛠️ Tools You’ll Need

  • Make (Free Tier) — easier than Zapier for beginners
  • Gmail — for email triggers
  • Google Drive — for file storage
  • Notion (Free Tier) — for notes + databases

All free. No credit card. No overwhelm.

Step 1: Pick Your First Task

Open a blank doc. Write down 3 repetitive tasks you do weekly. Examples:

  • Save Gmail attachments to Google Drive
  • Log meeting notes in Notion
  • Post new content to social media

Pick the easiest one. We’ll automate: “Save Gmail attachments to Google Drive.”

Step 2: Sign Up for Make

Go to make.com → “Sign Up” → Use Google or email.

Choose “Free plan” — 1,000 ops/month is plenty to start.

Verify email → Log in.

Step 3: Build Your First Scenario

Click “Create a new scenario” → Name: “Gmail → Google Drive”.

Click “Add your first module” → Search “Gmail” → Select “Watch Emails”.

Click “Add” → “Create a connection” → Log in to Gmail → Allow permissions.

Step 4: Configure Gmail Trigger

In “Watch Emails” module:

  • Connection: Your Gmail
  • Label: Leave blank (or create label “ToSave”)
  • Limit: 1

Click “OK” → Click “Run once” → Check green checkmark.

Step 5: Add Google Drive Action

Click “+” → Search “Google Drive” → “Upload a File”.

“Create a connection” → Log in → Allow permissions.

In “Upload a File” module:

  • File name: {{2.filename}}
  • File content: {{2.content}}
  • Folder: Choose or create “Saved Attachments”

Click “OK”.

Step 6: Activate & Test

Click “Turn scenario ON” → Set schedule: “Every 15 minutes”.

Send yourself a test email with attachment.

Wait 15 min → Check Google Drive → File appears. ✅

You just automated your first task.

✅ Results After 7 Days

  • 5+ hours saved
  • 0 manual file saving
  • Confidence to build next automation
